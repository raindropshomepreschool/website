# Raindrops Home Preschool — Modern Website

## Files in this package

| File | Purpose |
|------|---------|
| `index.html` | Main website (modern redesign) |
| `robots.txt` | Tells search engines what to crawl |
| `sitemap.xml` | Helps Google index the site faster |

## Deployment to Hostinger

1. Log into Hostinger File Manager
2. Navigate to `public_html`
3. Upload all 3 files (`index.html`, `robots.txt`, `sitemap.xml`)
4. Overwrite/replace the existing `index.html`
5. Your logos (RaindropLogo.png, KidLogo.png, HandLogo.png) already exist there — no need to re-upload

## After uploading

- Submit the sitemap to Google Search Console: https://search.google.com/search-console
  - Add property → enter https://raindropshomepreschool.com
  - Go to Sitemaps → enter `sitemap.xml` → Submit
